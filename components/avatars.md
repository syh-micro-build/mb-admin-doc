## Avatars 属性<span id="Avatars"></span>

| 属性 | 说明 | 类型 | 可选值 | 默认值 |
| ---- | ---- | ---- | ---- | ---- |
| size | 头像尺寸 | `ComponentSize、number` | - | - |
| max | 最大展示个数 | `number` | - | 5 |
| data | 头像数据，[详见](#data) | `AvatarItem[]` | - | - |
| showTooltip | 是否展示名称tip | `boolean` | - | true |

### data<span id="data"></span>

| 属性 | 说明 | 类型 | 可选值 | 默认值 |
| ---- | ---- | ---- | ---- | ---- |
| url | 头像图片地址 | `string` | - | - |
| name | 名称，非必填 | `string` | - | - |
